# 功能清单与优先级（Feature List）

## 1. 文档目标
明确V1 Demo的功能范围、优先级与验收标准，确保实现节奏聚焦在“学生决策效率提升”这一核心价值上。

## 2. 优先级定义

- P0（Must Have）：不做则无法体现产品核心价值。
- P1（Should Have）：显著提升体验和说服力，建议在V1完成。
- P2（Could Have）：加分项，可作为演示扩展或后续迭代。

## 3. 功能总览

| 模块 | 功能项 | 优先级 | 价值目标 |
|---|---|---|---|
| 首页发现 | 高校切换、场景入口、AI入口、推荐Feed | P0 | 让用户快速进入决策链路 |
| AI帮我选 | 对话输入、推荐结果、换一批 | P0 | 30秒内给出可行动候选 |
| 餐厅详情 | 基础信息、学生标签、评价摘要、行动按钮 | P0 | 促成导航/收藏/分享转化 |
| 地图浏览 | 高校锚点地图、点位展示、筛选 | P1 | 增强空间决策与可达性判断 |
| 榜单攻略 | 热门榜、低价榜、新店榜 | P1 | 提供无明确意图时的发现入口 |
| 个性化反馈 | 喜欢/不喜欢、偏好沉淀 | P1 | 建立推荐学习闭环 |
| 社交增强 | 二选一PK、分享投票 | P2 | 提升传播与协同决策效率 |
| 成就体系 | 美食足迹、打卡徽章 | P2 | 提升留存与趣味性 |

## 4. P0 功能清单（必须实现）

### 4.1 首页发现

功能点：

- 学校选择器（3-5所高校）
- 场景化入口（一个人吃/约会/聚餐/夜宵/穷鬼快乐餐）
- AI推荐入口卡片（突出主CTA）
- 推荐餐厅卡片流（含步行时长与同校热度）

验收标准：

- 首屏在移动端可完整显示学校选择、AI入口和至少3个场景卡片。
- 场景入口点击后，能进入对应结果列表或推荐页。
- 切换学校后，卡片中的步行时长与推荐数据同步刷新。

关键指标：

- 首页激活率（点击场景或AI入口）
- 场景入口点击率

### 4.2 AI帮我选

功能点：

- 自然语言输入（预算/人数/口味/距离/场景）
- 返回Top3候选餐厅
- 每个候选展示推荐理由（预算匹配、距离、同校热度）
- “换一批”能力

验收标准：

- 输入任意有效需求后3秒内返回推荐结果（模拟数据环境）。
- 推荐结果至少包含3个可点击候选项。
- 每个候选至少有2条可解释推荐理由。

关键指标：

- AI提交率
- 推荐采纳率（点击详情或行动按钮）

### 4.3 餐厅详情页

功能点：

- 基础信息（地址、营业时间、人均、联系方式）
- 学生标签与评价摘要
- 必点菜推荐
- 行动按钮（导航/收藏/分享）

验收标准：

- 从推荐卡片可直达详情页，且核心信息首屏可见。
- 行动按钮可触发对应行为（可为模拟跳转/提示）。
- 返回上一页时保留筛选条件与滚动位置。

关键指标：

- 详情到行动转化率
- 导航点击率

## 5. P1 功能清单（建议实现）

### 5.1 地图浏览模式

功能点：

- 以当前学校为中心展示餐厅点位
- 支持按品类/价格/步行时长筛选
- 地图与列表联动（点位点击高亮卡片）

验收标准：

- 切换学校时地图中心点同步切换。
- 筛选项变化后点位数量与列表同步变化。

### 5.2 榜单与攻略

功能点：

- 本周热门榜
- 穷鬼快乐榜（低价高性价比）
- 新店探索榜

验收标准：

- 每个榜单至少展示10条可点击条目。
- 榜单条目可进入统一详情页。

### 5.3 个性化反馈闭环

功能点：

- 推荐结果“喜欢/不喜欢”反馈
- 基于反馈的轻量偏好记录（本地或Mock层）

验收标准：

- 用户完成反馈后有即时提示。
- 再次请求推荐时可体现偏好方向变化（例如减少不喜欢菜系）。

## 6. P2 功能清单（加分项）

### 6.1 二选一PK

- 两家餐厅统一维度对比（预算、距离、评分、场景匹配）。
- 输出“更推荐哪家+原因”。

### 6.2 分享投票

- 生成可分享卡片给室友或社团群。
- 被分享方可快速查看候选餐厅关键信息。

### 6.3 成就与足迹

- 记录“已吃过”餐厅地图点亮。
- 达成里程碑后显示徽章。

## 7. 非目标范围（V1不做）

- 真实登录与学生认证体系
- 完整UGC发布与审核系统
- 真实支付/外卖下单闭环
- 商户后台和商业化结算系统

## 8. 开发排期建议（面向测评）

1. 第1阶段（1-1.5小时）：完成P0主链路页面与基础数据打通。
2. 第2阶段（1-1.5小时）：补齐地图/榜单与细节交互。
3. 第3阶段（0.5-1小时）：反馈闭环、文案润色、演示脚本联调。

## 9. 评测呈现建议

- 演示时优先走P0闭环：首页 -> AI推荐 -> 详情 -> 导航。
- 口头强调“为什么先做这些功能”：直指北极星指标“决策转化率”。
- 对P1/P2明确说明“已规划可扩展”，体现产品优先级判断。
