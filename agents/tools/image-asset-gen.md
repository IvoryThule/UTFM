# Tool: 图片资源生成
## 工具概述
为 Demo 提供 Logo、餐厅图、头像、空状态插画等资源方案，优先稳定可控。
## 资源建议
1. Logo：Emoji + 文本（低成本高可用）
2. 餐厅图：优先本地占位图，其次外链随机图
3. 头像：文字头像或 DiceBear
4. 空状态：Emoji组合 + 友好文案
## 目录约定
- public/favicon.ico
- public/images/*
- src/frontend/components/ui/FoodImage.tsx
## 餐厅图策略
### 优先方案：本地/渐变占位
- 不依赖外网
- 样式统一
- 不会因外链失效破图
### 备选方案：外部随机图
- 可快速提升真实感
- 需处理加载失败
## 统一图片组件要求
FoodImage 组件必须支持：
- loading 骨架/占位
- error 兜底
- lazy loading
- 分类色彩占位
## 验收标准
- [ ] 餐厅卡片均有可见封面
- [ ] 图片失败时有兜底占位
- [ ] 空状态页面有友好插画/文案
- [ ] 头像有默认方案
- [ ] 移动端加载无明显闪烁
