# Workflow: 打磨迭代
## 触发条件
核心功能开发完成后执行。
## 打磨维度
### 1. 视觉一致性
- 色彩、圆角、阴影、字号、间距统一
- shadcn/ui 风格一致
### 2. 文案调性
- 学生语境
- 行动导向
- 空状态友好
### 3. 交互体验
- 可点击元素有反馈
- 页面切换平滑
- AI打字机节奏自然
### 4. 移动端适配
- 360/375/393/430 逐一检查
- 键盘遮挡、底部安全区、横向溢出
### 5. 数据真实感
- 价格、距离、标签、评价一致性
### 6. 性能
- 首屏加载可接受
- 图片懒加载
- 控制台无严重错误
## 推荐顺序
1. 先视觉
2. 再交互
3. 再文案
4. 最后性能与细节
## 完成标准
- [ ] 主链路可稳定演示
- [ ] 页面视觉与交互统一
- [ ] 移动端关键机型通过
- [ ] 无阻断问题
